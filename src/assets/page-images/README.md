# Page Image Folders

Drop image files into these folders to show them on each page:

- `src/assets/page-images/about/`
- `src/assets/page-images/paintings/`
- `src/assets/page-images/code-art/`
- `src/assets/page-images/robots/`
- `src/assets/page-images/community/`
- `src/assets/page-images/face-value/`

Supported file types: `png`, `jpg`, `jpeg`, `webp`, `gif`, `avif`.

If you add brand new files while `npm run dev` is running and they do not appear immediately, restart the dev server once.
